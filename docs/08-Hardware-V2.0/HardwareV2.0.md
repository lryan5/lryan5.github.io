## Ideas for Hardware Improvement

If I were do redo this project again, there would be several things in which I would improve upon. The first, would be the spacing and labeling of the jumpers and components on my PCB. As shown in PCB pictures in the PCB Design page, I was not able to finish soldering my components to my PCB board. Part of this is due to how much longer my team's PCB shipment took to ship compared to other. Also, the positioning of my parts on the board were pretty cramped. This made it difficult to solder pieces onto the board that were close together. To fix this issue in the future, I would plan out how I would solder my parts to the board and position them further apart. I was only able to solder my power supply onto my board due to me believing that I messed it up as the connection that was supposed to by supplying 5V was instead supplying 9V. This happened because I closed a jumper that would connect my 5V and 9V together if my teammates and I were sharing power amongst each other. To fix this, I would label important jumpers using the silkscreen so I would be more knowledgable on which ones might cause my system to not function.

Another change that I would make to improve the hardware would be 3D printing a housing for my IR LED and phototransistor. Currently, the components for the IR sensor would be attached to the PCB through screw terminals and wires. This wouldn't work very well for a product as the transmitter and receiver of the sensor wouldn't be in alignment. Creating a housing for both parts would solve this issue and make it easier to mount on a trash can.
