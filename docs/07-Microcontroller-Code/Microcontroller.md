## Overview

This is the code used in my IR sensor subsystem to interact with the other subsystems and the user. This code allows my subsytem to inform the speaker and motor subsystems to activate when the user waves their hand in front of the IR sensor. Also, when the weight subsystem detects there is too much trash in the trashcan, my subsystem will inform the user through a red LED.
