## Lessons Learned

1. Start and finish PCB design earlier than you need do <br>

2. Keeping updated files on all of the computers you use is very important

3. Soldering surface mount components is far easier with a heat gun.

4. Communication protocols and data types being sent need to defined asap

5. Test in increments to avoid costly mistakes

6. Read the whole datasheet before confirming components

7. Try to standardize components between subsystems

8. Completing steps early leaves time for refinement
